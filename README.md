# Smart-Table-Lamp
智能护眼台灯（Arduino）  基于 Arduino 的多功能智能护眼台灯，集成环境光感应、人体感应、坐姿检测、学习/考试计时等功能，支持自动调光与蓝牙控制，提供舒适、健康的照明体验。  主要功能  • 自动模式：根据环境亮度自动开关灯，并平滑无级调光，不刺眼  • 学习模式：可设定学习时长与休息提醒，自动计时与灯光控制  • 考试模式：定时专注模式，时间结束自动提醒并恢复待机  • 手动模式：支持按键/蓝牙指令手动开关灯  • LCD 实时显示：环境亮度、距离、人体感应与工作状态  • 状态完全同步：屏幕显示与灯光状态一致，无逻辑冲突  适用场景  课程设计、开源硬件项目、智能照明、Arduino 综合实训作品
