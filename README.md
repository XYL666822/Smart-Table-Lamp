智能护眼台灯 - Smart Eye-Care Desk Lamp

基于 Arduino 的多功能智能照明项目，集成环境光自适应、学习/考试计时、坐姿检测与蓝牙控制，适合课程设计、开源实训与智能硬件作品。

功能介绍

• 🌞 自动调光模式
根据环境光线强度自动开关灯，并实现平滑无级亮度调节，不频闪、不刺眼。

• 📚 学习模式
支持自定义学习时长与休息提醒，定时结束自动提示并恢复待机。

• ⏱️ 考试模式
专注计时模式，时间到自动声光提醒，状态同步无错乱。

• 🎮 手动控制模式
支持串口/蓝牙指令手动开关灯与切换模式。

• 📺 LCD 实时显示
显示环境光值、距离传感器、人体感应状态与当前工作模式。

• 🔒 状态严格同步
修复灯光与屏幕显示不一致问题，运行稳定无逻辑 Bug。

硬件清单

• Arduino Uno

• 1602 LCD 屏

• LDR 光敏电阻

• HC-SR04 超声波传感器

• PIR 人体红外传感器

• WS2812 灯带

• ESP8266（可选，蓝牙/无线控制）

控制指令

• A — 进入/退出自动模式

• M — 手动开关灯

• S — 进入学习模式

• E — 进入考试模式

适用场景

• 单片机课程设计

• Arduino 综合实训项目

• 智能照明 / 物联网入门作品

• 开源硬件学习与演示
